## Catatan Scraping

### Kolom kritis untuk buzzer detection
```
X:         account_created_at, followers_count, following_count, tweet_count, source (client app)
Instagram: account_created_at, followers_count, following_count, post_count, biography
YouTube:   channel_created_at, subscriber_count, total_video_count
Facebook:  page_followers_count, page_likes_count (untuk page), user_id (untuk personal)
```

### Kolom kritis untuk interaction network (SNA)
```
X:         in_reply_to_user_id, retweeted_tweet_id, quoted_tweet_id, mentions
Instagram: comment_user_id → post_username (reply ke siapa), mentions dalam caption
YouTube:   comment_author_id → channel_id (video owner), reply_to_comment_id
Facebook:  comment_user_id → page_id / post user_id
```

### Kolom kritis untuk co-hashtag network
```
X:         hashtags (dari entities)
Instagram: hashtags (parse dari caption)
YouTube:   tags (dari video metadata)
Facebook:  hashtags (parse dari message)
```

### Kolom kritis untuk DNA (aktor + stance + konsep)
```
Semua platform: text/caption/comment_text + username/channel_name + created_at
```
